# Signature-Verification-System
A simple web-based Signature Verification System built using **Flask**, **Python**, **OpenCV**, **HTML**, **CSS**, and **JavaScript**.
The project allows users to upload two signature images and compares them using image processing techniques to determine whether the signatures match or not.

## 📌 Project Idea
This system was designed as a simple assistance tool for bank employees to help verify client signatures quickly and reduce manual effort during the verification process.

The project focuses on:
* Simplicity
* Easy understanding
* Practical implementation
* Basic image processing concepts

## 🚀 Features
* Upload two signature images
* Image preview before comparison
* Signature preprocessing using OpenCV
* Grayscale conversion and binarization
* Noise removal
* Feature extraction using contours and Hu Moments
* Signature comparison using Euclidean Distance
* Match / Not Match result display
* Modern responsive user interface
* File type validation and error handling
